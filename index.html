TODDLER FIRST WORDS

Welcome to Toddler First Words

Kids need to be taught some basic things while they are young. They need to learn about animals, birds, and other such basic things so that they can figure out on their own when they see them in real life. This project is about learning the above mentioned. It has different categories to choose from. Each category has different pictures which will help them learn easily as to seeing is learning and along with each picture, there is a basic information on them.
